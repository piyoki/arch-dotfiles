## Submodules

Add new submodules

```bash
yadm submodule add git@github.com:<owner/repo>.git modules/<module_name>
# e.g
# yadm submodule add git@github.com:yqlbu/dot-nvim.git ~/.config/nvim
```

Sync submodules

```bash
yadm submodule update --init --recursive
```
